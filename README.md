## Requirements

* **Flask**

`pip install Flask`

* **Werkzeug**

`pip install Werkzeug`

* **OpenCV**

`pip install opencv-python`

## Usage

#### Step #1 :

* Clone the Repo. 

#### Step #2 :

* Install the [Requirements](#requirements)

#### Step #3 :

* Fire up the terminal, and execute this piece of snippet :

`python app.py`
